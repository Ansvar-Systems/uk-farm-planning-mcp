# Changelog

## 0.1.0 (2026-04-03)

Initial release.

- 11 tools (3 meta + 8 domain)
- Crop rotation guidance (13 pairs): suitability, disease breaks, blackgrass risk, yield impact
- Gross margin benchmarks (13 enterprises): arable + livestock, AHDB Farmbench-style 2024/25
- Agricultural tax rules (8 topics): MTD, farmers averaging, capital allowances, SBA, VAT, partnership, IHT
- Agricultural Property Relief guidance (10 scenarios): 100%/50% rates, occupation tests, clawback, farmhouse proportionality
- Tenancy law rules (10 entries): AHA 1986 vs ATA 1995, succession, rent, compensation, Gladstone v Bower
- Diversification guidance (6 activities): Class Q/R/S, camping, solar, planning requirements
- Break-even calculator using gross margin benchmarks
- FTS5 full-text search across all data
- SQLite database, stdio + Streamable HTTP transport
